GPTeverywhere

GPTeverywhere is a Chrome extension that brings the power of OpenAI's GPT API to your browsing experience. With this extension, you can generate context-aware text suggestions in any text input field on websites. To use it, simply type $gpt followed by your input text and press CTRL + ALT. The extension will replace your input with an AI-generated response based on your prompt, up to 150 tokens in length.

To get started, enter your OpenAI API key in the field below and click "Save". Please note that by using this extension, you agree to the OpenAI API Terms of Service. The creators of this extension are not responsible for any misuse of the API or any damages resulting from its use.
